    EVA

    一款因作者需求而生的一款极简风谷歌扩展!

    扩展商店地址

    logo作者

    Peter Qian(钱学毅)

    功能

    • 时间戳转换
    • 图片转base64
    • base64转图片
    • 字符串编解码(md5, base64)

    使用方法

    直接点击扩展图标会弹出主页面

    页面截图

    离线安装方法

    1. yarn 安装依赖
    2. yarn dev 将编译好的 build 文件夹拖拽进谷歌浏览器中。

    你也可以直接在Releases直接下载最新的zip包，解压后直接将文件夹拖拽到chrome扩展页面。
