    • 2020年度总结

    总结

    2020年这一年， 因为人员调度问题， 导致只有我这一个开发在杭州。 所以，来自同事和办公室政治的压力微乎其微， 绝大多数的压力只是来自于项目的时间节点和交付（这个对于我这个人的性格不是很讨厌）。
    ex：就是做归档做的我很头疼。 特别那个归档测试， （黑脸表情）。

    研发部经理很nice， 给了我很大的自由空间， 自己处理自己的工作确实很爽。 我本人愿意为自己承担的事情负责。

    优点

    1. 技术自由。 技术栈的自己选择、 技术部分的不限制、因为自己对需求， 所以可以对固定的需求， 可以自己制定相对友好的解决方案等等。
    2. 时间自由， 今年在领导的帮助协调下， 即完成了工作， 也完成了学业。 而且工作的时间很有弹性。
    3. 报销容易。 对报销的审核很是友好，没有遇到质疑。 这个很nice。
    4. 今年白嫖回来一个4卡服务器对我帮助非常大， 明年可以在相对方向深入不少。 解决了我很大的硬件问题。

    缺点

    1. 薪水低。 明年我的目标可是理财、学习和攒钱， 哈哈。
    2. 今年11-12月份期间，任务繁重， 希望明年如果有长期的多任务环节（工作量在1.6人加班才能干完的情况）， 如果可以招聘技术强劲的同事（可以满足可以接手依图开发栈的选手）。 一起合作那更好了。 当然是在多任务的前提下。
    3. 德清住宿确实差劲， 但现在已经搬到台州， 台州这边是满意的。
    4. 餐饮费报销， 开发没有餐饮费报销， 我希望可以给我自己每月一次餐饮报销额度。
